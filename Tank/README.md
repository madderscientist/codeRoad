# 幻影坦克
js和py版
- 创建日期：JS: 2022年5月9日，22:22:41，py: 2023年1月8日，22:31:38
- 起因：QQ机器人不能光明正大发色图。一开始用的jsbot，写了js版；后来服务器到期忘备份了，转用python bot，写了python版。
