# 辉光管时钟桌面
- 创建：2023年2月14日，22:51:07
- 目的：学习Win32编程。
- 参考资料：https://www.bilibili.com/video/BV1HZ4y1978a
- 动机：看到了辉光管时钟但是没钱，就想装一个虚拟的在桌面上。没有wallpaper engine，只能自己调用win32 api写。
- 使用：在关闭一切桌面应用后执行exe。右下角托盘区可以退出应用。
- 成果：学会了创建windows窗口；学会了篡改桌面；学会了创建托盘图标。
